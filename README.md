### Run The App

To get started running the react web app...

1. Clone the repo
    ```sh
    git clone https://github.com/tobylangdon/mission-zero.git
    ```
2. Install NPM packages
    ```sh
    npm install
    ```
3. Start localhost:3000 by running react script start
    ```sh
    npm run start
    ```
